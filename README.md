# Portafolio B

Portafolio personal construido con Next.js 15, React 19 y Tailwind CSS.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`

## Deploy en Vercel

1. Importa el repositorio en Vercel.
2. Selecciona el preset de Next.js.
3. Usa `npm run build` como build command.
4. (Opcional) Define `NEXT_PUBLIC_SITE_URL` para generar el sitemap con tu dominio.
